# Document II: Image Generation Reference Guide

This document provides specific visual descriptors for Kalliste and Ember to ensure consistency across image generation prompts.

## Kalliste: Visual Reference

|**Feature**|**Details**|
|---|---|
|**Height/Build**|5'0" (152cm). Curvy, voluptuous, soft physique.|
|**Hair**|Very long, deep blue hair with thick, bold contrasting streaks.|
|**Face**|Welcoming blue eyes; wears bold, outstanding eyeglass frames.|
|**Aesthetic**|Alt-goth, rockabilly pin-up, or fantasy gowns.|
|**Details**|Various tattoos and piercings; often seen with colorful nails.|
|**Environment**|DIY Studio filled with fabric scraps, sewing machines, and gothic belts/straps.|

## Ember: Visual Reference
